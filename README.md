# Health Calculator

Health Calculator is a tool designed to help users calculate their total daily energy expenditure (TDEE)
using the Harris-Benedict equation.

![Screenshot](reposcreenshot.PNG)

## Getting started

To get started:
```
npm install
npm run start
```

Production environments:
```
npm run build
```
